# NatureTunnel图形化客户端启动文档

## 一、介绍

- 本文档将一步步向您介绍如何部署并使用NatureTunnel内网穿透服务图形化客户端。

## 二、操作步骤

### 准备工作

- [下载](https://console.nrtun.com/release) 适合您操作系统架构的最新版本的NrtunLauncher图形化客户端安装程序（例如NatureTunnel.Launcher Setup 1.3.0.exe），并运行。

:::warning
由于程序涉及流量转发，可能会被部分杀毒软件报毒，请将客户端添加至信任区防止进程被杀
:::

### 启动客户端

- 安装完成后，启动NrtunLauncher图形化客户端。

- 登录客户端后，您将看到一个形如下图的界面。

![客户端页面](https://docs-assets.bucket.uciu.cn/client-home-page.png)

### 启动隧道

- 在上部导航栏点击进入隧道管理，您将看到一个形如下图的界面。

![隧道管理页面](https://docs-assets.bucket.uciu.cn/client-tunnel-list-page.png)

- 不出意外的话，您可以在这里看到您事先在浏览器控制台上创建的隧道。

- 如图所示，点击启动隧道。

:::tip
如果一切顺利，您可以在控制台查看到隧道已启动并且成功访问您的服务。
:::

### 查看日志

- 在上部导航栏点击进入日志，您将看到一个形如下图的界面。

![日志页面](https://docs-assets.bucket.uciu.cn/client-runlogs-page.png)

- 如果您的隧道启动失败，请将这**部分日志截图**，并发送至 [QQ用户交流群](https://qm.qq.com/q/yvjuIQj1LM) 内寻求帮助。
