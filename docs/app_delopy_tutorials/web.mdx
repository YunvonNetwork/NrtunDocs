# Web 服务穿透

:::warning
出于合规性要求，我们已不允许TCP协议的穿透进行HTTP/HTTPS协议的服务，因此请使用HTTP/HTTPS协议的穿透来暴露HTTP/HTTPS服务。
:::

:::tip
此文档尚在编写中
:::

## 介绍

本文档将一步步向您介绍如何使用NatureTunnel内网穿透服务将本地的HTTP/HTTPS服务暴露到公网。

- 本教程适用于用户将本地的 HTTP/HTTPS 服务穿透到公网的场景，如 博客、NAS、服务器面板等。
- 对于暴露到公网的敏感服务，请自行做好安全防护措施，我们不对穿透服务的安全性负责。

## 操作步骤

### 1. 创建隧道

- 在浏览器中打开 [控制台](https://console.nrtun.com/)，登录后进入顶栏导航 [工作台-创建隧道](https://console.nrtun.com/tunnel/create) 页面。
