# 关于大流量服务的定义

## 介绍

- 本文档旨在帮助您了解我们对大流量服务的定义。
- 我们允许您使用我们的平台搭建大流量服务，但是需要使用节点介绍内明确表明允许搭建大流量服务的节点，从而防止大量占用低带宽节点的资源影响其他用户正常使用。

## 特征

### 通常情况下，我们将具有以下特征的服务定义为大流量服务：

- 10分钟内，平均带宽高于10Mbit/s的。
- 峰值带宽超过100Mbit/s以上的。
- 传输单个文件超过100MiB的。
- 搭建图床、文件分享、网盘类型程序的。
